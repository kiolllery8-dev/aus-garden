### 系列定位說明

- **A 類（蠟燭製作）**：核心主力，視覺傳播力最強
  → 上鏡好看、成品漂亮、社群照片素材多，是引流與品牌曝光的王牌
  → 行銷優先級最高，也是新客入門首選

- **B 類（精油調香）**：知識感 + 體驗感，客單較高
  → 需要老師深度引導，客人願意付更高單價
  → 適合經營「深度回客」與「香氣知識型」內容
