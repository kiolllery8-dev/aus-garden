# docs/ ｜ AUS GARDEN 品牌憲法

## 這個資料夾是什麼？

這裡存放 AUS GARDEN 的**靜態事實**——品牌的「憲法」。
內容一旦確立，就不會每天改變，像一本公司手冊。

## 特性

- **靜態**：內容穩定，變動頻率低（月/季為單位）
- **事實**：品牌的既定資訊、規範、標準
- **唯讀**：代理人只讀取，不主動修改
- **權威**：當記憶與事實衝突時，以 docs/ 為準

## 建議收錄內容

- `brand-guidelines.md`    品牌識別、Logo、配色、字型規範
- `voice-tone.md`          品牌語氣、文案風格、禁用詞
- `products.md`            商品目錄、規格、定價（完整版）
- `courses.md`             課程完整目錄、時長、定價
- `pricing.md`             價格表（B2C / B2B / 顧問服務）
- `target-audience.md`     目標客群輪廓、需求、痛點
- `brand-story.md`         品牌故事、創辦理念
- `legal-disclaimers.md`   法規聲明、免責條款
- `contact-info.md`        聯絡資訊、營業時段

## 使用規則

1. 只有**品牌負責人**可以修改這裡的檔案
2. 修改前請在檔案頂部更新版本號與日期
3. 重大變更需同步通知所有代理人（更新 CLAUDE.md）
